# 로컬마켓 프론트엔드 데모 (gangdong_market)

강동구 상점가 대상 로컬마켓 서비스의 프론트엔드 데모 프로젝트입니다.
현재 버전은 고덕2동 골목형상점가를 중심으로, 멤버십-단골-커뮤니티-혜택 흐름을 시연하는 MVP UI 프로토타입입니다.

- 프로젝트 성격: 프론트엔드 데모 (Vite + React + TypeScript)
- 백엔드 연동: 미연동 (정적 목데이터 기반)
- 관련 백엔드(작업중): ph-cms

---

## 1. 서비스 정의 (최신 사업기획 반영)

최근 문서 "로컬마켓 사업기획 방향 정리" 기준으로 본 서비스의 핵심 정의는 아래와 같습니다.

> 로컬마켓은 지역 상점가별 멤버십과 가게별 단골 관계를 기반으로,
> 주민-상인회-개별 점포가 소식과 혜택을 주고받는 웹 기반 상권 커뮤니티 운영 서비스다.

즉, 본 서비스는 지도 대체 앱이 아니라 관계 기반 운영 플랫폼을 지향합니다.

- 상점가 멤버십: 상권 단위 관계 형성
- 가게 단골: 점포 단위 관계 형성
- 커뮤니티/단골 소식: 소통 채널
- 알림/쿠폰/스탬프: 재방문 유도 장치
- 관리자 대시보드: 운영 성과 관리

## 2. 구현 범위 요약 (현재 코드 기준)

### 2-1. 구현된 주요 화면

- 포털 홈
  - 상점가 카드 목록
  - 시즌 배너
  - 빠른 진입 카드
- 시장 홈 (고덕2동 중심)
  - 메인/상점가 지도/피드 탭
  - 멤버십 상태별 UI 분기
  - 커뮤니티 미리보기, 이벤트, 점포 목록
- 가게 상세
  - 멤버 여부/단골 여부에 따른 CTA 분기
  - 단골 소식 잠금/해제
  - 쿠폰/스탬프 섹션
- 피드
  - 커뮤니티 피드
  - 단골 소식 피드 + 가게 필터 + 상세 진입
- 혜택
  - 스탬프투어 보드
  - 쿠폰함 + 쿠폰 제시 화면
- 마이페이지
  - 멤버십 상태
  - 단골 가게 목록
  - 알림 토글
  - 운영 대시보드 진입
- 운영 화면
  - 상권 운영 대시보드
  - 상인회 대시보드
  - 구청 대시보드
  - 상점 대시보드
- 알림 화면
  - 알림 목록
  - 알림 클릭 시 가게 상세 이동

### 2-2. 현재 동작 방식

- 상태 관리: React Context 기반 전역 스토어
- 데이터: src/data.ts 정적 데이터
- 인증/권한: 데모 상태 토글 수준 (실제 인증 없음)
- 서버 통신: 없음

## 3. 문서 대비 기능 매핑

아래는 docs 기준 요구사항과 현재 데모 구현의 매핑입니다.

## 4. 실행 방법

### 요구사항

- Node.js 18+
- pnpm 10+ (권장)

### 설치

```bash
pnpm install
```

### 개발 서버 실행

```bash
pnpm dev --host
```

기본 접속 주소(예시):

- Local: http://localhost:5174/
- Network: http://<내_로컬_IP>:5174/

### 빌드

```bash
pnpm build
```

### 빌드 결과 미리보기

```bash
pnpm preview
```

## 7. 디자인/UX 방향

본 데모는 다음 문서의 방향을 반영합니다.

- 서비스 기획: 멤버십/단골 중심의 폐쇄형 상권 커뮤니티
- 화면 기획: S-01 ~ S-14 단계형 사용자/운영자 흐름
- 사업 기획: SI 납품형이 아닌 반복 도입형 서비스 모델

핵심 원칙:

- 검색보다 참여 유도
- 점포 정보보다 상권 경험 중심
- 운영자가 직접 갱신 가능한 단순한 운영 구조

## 8. 백엔드 연동 계획 (ph-cms)

현재 프론트는 정적 데이터 기반 데모이며, ph-cms와 연동 시 아래 도메인을 우선 연결합니다.

1. Auth/회원
   - 로그인, 멤버십 가입/해지
2. Market/Store
   - 상점가, 점포, 카테고리, 지도 좌표
3. Membership/Dangol
   - 상점가 멤버, 가게 단골 관계
4. Feed
   - 커뮤니티 글/댓글/반응
   - 단골 소식
5. Benefits
   - 쿠폰 발급/사용
   - 스탬프투어 미션/완주
6. Notification
   - 상권/가게별 발송, 읽음 상태
7. Admin/Report
   - KPI 집계, 대시보드, 리포트
